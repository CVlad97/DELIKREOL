# Contribuer à DELIKREOL

## Prérequis
- Node.js 18+
- npm ou yarn
- Compte Supabase (mode demo disponible sans)

## Installation locale
```bash
git clone https://github.com/CVlad97/DELIKREOL.git
cd DELIKREOL
npm install
cp .env.example .env
npm run dev
```

## Workflow Git
- Branch depuis `main`
- Nommage : `feat/nom-feature`, `fix/nom-bug`, `docs/nom-doc`
- PR avec description claire
- Tests requis avant merge

## Standards de code
- TypeScript strict
- ESLint + Prettier
- Commits conventionnels (feat, fix, docs, chore)
