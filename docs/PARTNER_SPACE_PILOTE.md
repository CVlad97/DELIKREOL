## Accès
0 € — modification autonome gratuite pendant la phase pilote.
Le partenaire peut modifier lui-même :
- bio
- photos
- plats
- prix net vendeur
- composition
- allergènes
- horaires
- zone
- mode retrait / relais / livraison
- disponibilité

## Statuts des modifications
Chaque modification passe par :
1. Brouillon — le partenaire édite
2. Soumis — envoyé pour validation
3. À valider — en attente admin
4. Approuvé — publié sur le site
5. Refusé — retour pour correction

## Tarification plats
Le partenaire indique son prix net souhaité.
DELIKREOL calcule le prix client : `prix_net_vendeur / 0.85`
Arrondi commercial : 0,50 / 0,90 / euro supérieur.

Exemple :
- Prix net vendeur : 10,00 €
- Prix client DELIKREOL : 11,90 € (10,00 / 0,85 = 11,76 → arrondi 11,90)
- Commission DELIKREOL : 1,90 €
