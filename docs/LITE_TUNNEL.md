# DELIKREOL Lite premium - Tunnel principal

Ce document decrit le parcours principal en mode Lite premium, stable et WhatsApp-first.

## 1) Home
- Hero et CTA principaux
- Acces aux vendeurs pilotes
- Catalogue avec recherche, filtres et tri

## 2) Vendeur / Menu
- Acces via cartes partenaires ou deep-link `?vendor=Nom%20Vendeur`
- Vue vendeur enrichie (histoire, promesse, delai indicatif)
- Produits filtres par vendeur

## 3) Panier / Checkout
- Ajout produit -> panier
- Checkout structure (mode, planification, infos client, recap)
- Mode entreprise disponible (champs supplementaires)
- Validation finale via WhatsApp

## 4) WhatsApp (confirmation)
- Message genere avec: vendeurs, produits, mode, planning, infos client
- Mention explicite: "creneau souhaite soumis a confirmation"

## 5) Suivi
- Page de suivi disponible mais non temps reel
- Informations presentent l'etat de facon honnete (manuel/pilote)

## Limites connues
- Paiement non integre
- ETA indicatif, confirmation manuelle
- Fidelite en mode pilote (activation manuelle)
